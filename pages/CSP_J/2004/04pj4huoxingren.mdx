#### 核心思想：
本题的难点是阅读理解。通读后发现，题目的本质是**全排列**，可以直接调用 **next_permutation()** 进行。

#### 题解代码：
```cpp showLineNumbers filename="next_permutation()" {11}
#include <bits/stdc++.h>
using namespace std;

const int N = 10005;
int n, m, a[N];

int main()
{
    scanf("%d %d", &n, &m);
    for(int i = 0; i < n; i++)  scanf("%d", &a[i]);  // 读入外星人的数据
    for(int i = 0; i < m; i++)  next_permutation(a, a + n);  // 对a执行m次 next_permutation()
    for(int i = 0; i < n; i++)  printf("%d ", a[i]);
    return 0;
}
```
