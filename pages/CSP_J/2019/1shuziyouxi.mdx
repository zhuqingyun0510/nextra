#### 核心思想
本题的长度只有8，送分题，读入时逐一判断即可

#### 题解代码
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
	char c;
	int cnt = 0;
	
	for(int i = 0; i < 8; i++)
	{
		cin >> c;
		if(c == '1')	cnt++;
	}
	
	cout << cnt << endl;
	return 0;
}
```
